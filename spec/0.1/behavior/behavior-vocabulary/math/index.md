# Behavior Vocabulary — Math

This specification defines the v0.1 **Math operator family** for the Behavior Vocabulary.

This document is **Normative**.

---

## 1. Scope (v0.1)

v0.1 does not yet fully specify the Math operator inventory and semantics.

Until this section is expanded, implementers MUST treat Math operators as **unspecified** unless they are defined in a separate v0.1 normative spec.

---

## 2. Non-Goals (v0.1)

- No implicit numeric widening or host-specific numeric types.
- No reliance on JSON number edge cases.
