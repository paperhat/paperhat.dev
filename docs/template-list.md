## A Useful Mental Model

Most of these can be reduced to combinations of:

- **Content type** (text, media, data)
- **Authorship** (single, collaborative, anonymous)
- **Mutation** (read-only → fully editable)
- **Temporal structure** (static, feed, timeline, event-based)
- **Interaction** (none, comments, transactions, real-time)
